
SyncAnywhere Installation Guide
==============================

AUTOMATIC INSTALLATION:
1. Run the installer EXE
2. Follow the installation wizard
3. The service will start automatically
4. Use the desktop shortcut to check status

MANUAL CONFIGURATION:
1. Edit config.json and set your DSN name
2. Run: ServiceInstaller.exe install

MANAGING THE SERVICE:
- Check Status: ServiceInstaller.exe status
- Restart: ServiceInstaller.exe restart  
- Uninstall: ServiceInstaller.exe uninstall

MOBILE APP CONNECTION:
- Find your computer's IP address
- Use: http://YOUR_IP:8000
- Default pairing password: IMC-MOBILE

SUPPORT:
- Check logs in the 'logs' folder
- Service runs automatically on Windows startup
- Uninstall through Windows Add/Remove Programs
